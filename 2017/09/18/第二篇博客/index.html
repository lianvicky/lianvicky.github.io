图片裁切

jCrop

  1. 引入css文件
  2. 引入jquery
  3. 引入插件js
  4. 创建一个图片标签
  5. 在这个图片标签上调用 $(“#target”).jcrop()

设置项：

  1. setSelect : [x, y, w, h] 设置默认选区
  2. aspectRatio: 设置选区的宽高比
  3. bgColor
  4. bgOpacity
  5. maxSize minSize
  6. boxHeight
  7. boxWidth
